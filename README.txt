universa-3000

universa-3000 is a gravitation simulation.

Install:
1. Download cogaen3-java:
	link: http://github.com/divotkey/cogaen3-java
	compatible with commit 288223a90f7c3d7a20b571b12aa9cce823d303cc
2. add Cogaen Core and Cogaen LWJGL to build path:
	in eclipse: inport Projects and add Projects to build path.
3. set native library location:
	in eclipse: Libraries -> Native library location -> Edit -> 
		Workspace -> Cogaen LWJGL -> lib2xx -> *your platform*
4. have fun!

Controls:
drag mouse to drag view
'n' and 'm' keys to zoom
'v' and 'b' keys change the gamespeed
